Automatically adds the sponsornot badges to posts

== Description ==

Sponsornot Plugins automatically adds sponsornot badges to posts where you choose to display badges. Badges are linked to the sponsornot Badge page explaining the different sponsornot types: 
* O for no collaboration
* G for Gifts
* S for Sponsored reviews

== Installation ==


1. Upload the plugin files to the `/wp-content/plugins/sponsornot` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Use the Settings->sponsornot screen to configure the plugin.
4. select the sponsornot attribution on the edit post page.
